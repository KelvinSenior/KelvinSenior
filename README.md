# Sarfo Kelvin Senior — High-Ticket Landing Page

A premium landing page for marketplace and SaaS systems architecture. Positions Sarfo as a high-ticket architect specializing in revenue-generating platforms.

## Structure

**5 Clean Sections:**

1. **Hero** — Authority positioning with CTA to book strategy call
2. **What I Build** — Three service offerings (Marketplace Platforms, SaaS MVP Infrastructure, Custom Business Systems)
3. **Case Studies** — Three architected systems projects
4. **About** — Strategic positioning focused on monetization & efficiency
5. **Call To Action** — Final conversion section with Calendly link

## Setup

- Open `index.html` in a browser
- Update `https://calendly.com/sarfokelvinsenior/30min` if needed in the CTA section
- Replace the hero image at `assets/Photos/sarfo_kelvin_senior 2.jpg` if desired
